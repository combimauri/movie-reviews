# MovieReviews

This is a demo project that lists movies that can be reviewed. Its main purpose is to teach modern Angular concepts like smart and dumb components, along with the new input, output, and model signal APIs.

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.
